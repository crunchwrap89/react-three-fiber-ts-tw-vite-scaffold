# React-Three-Fiber scaffold

Scaffold / template for creating a new React, Typescript, Tailwind, Vite and Vitest app with React-Three-Fiber.

## Init from scratch

npx degit crunchwrap89/react-three-fiber-ts-tw-vite-scaffold

## Install

```bash
npm install
```

## Run Dev

```bash
npm run dev
```

## Run Tests

```bash
npm run test
```

## Lint

```bash
npm run lint
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
